  Applies a task list to a room, based on the `taskTemplateId` that you specify in the `taskList` request body.  **Note:** You must first create a task list template in the console. You cannot create a task list template by using the API at this time. To get the `taskListTemplateId`, use the 	TaskListTemplates: GetTaskListTemplates method.  ## Permissions  You must have the `canApplyTaskList` permission set to **true.**
