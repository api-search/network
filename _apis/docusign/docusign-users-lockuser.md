  Locks the account of the specified user. Use the `reason` property in the request body to specify why the account was locked.  ## Permissions  To lock a user who has a lower access level than you, you must have the `canManageMemberLowerAccessLevel` permission set to **true.** To lock a user who has the same access level as you, you must have the `canManageMemberSameAccessLevel` permission set to **true**
