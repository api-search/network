  Fetches route configuration from the management server. The Route Discovery Service (RDS) returns route configuration that maps incoming requests to specific upstream clusters based on request attributes such as path, headers, and query parameters. Route configurations define virtual hosts, route matching rules, and traffic management policies including retries, timeouts, and rate limiting.
