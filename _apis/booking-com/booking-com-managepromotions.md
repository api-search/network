description: >-
  Creates or updates promotional offers for a property. Supports multiple promotion types including basic deals (fully customizable), last-minute deals (available only close to check-in), campaign deals (seasonal events with fixed dates and minimum discounts), country rates (available to customers from specific countries), and mobile rates (available only on mobile devices). Each promotion specifies a discount percentage applied to the parent rate.
