description: >-
  Applies one or more updates to the document. Each request is validated before being applied. If any request is not valid, the entire request will fail and nothing will be applied. Some requests have replies to give you some information about how they are applied. Other requests do not need to return information; these each return an empty reply. The order of replies matches the order of requests.
