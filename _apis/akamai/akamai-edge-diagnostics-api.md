  Edge Diagnostics allows you to diagnose your server, DNS, and network problems
  from Akamai servers around the world.Once you extend your web content onto the
  Akamai edge network and apply various Akamai features to accelerate and
  manipulate content, you need to be able to troubleshoot any problems your
  users may encounter. With Edge Diagnostics API you can diagnose common
  problems you may experience when delivering content to your users, except for
  China CDN.
