  The Certificate Provisioning System (CPS) provides full life cycle management
  of SSL/TLS certificates for your Akamai Secure Delivery Network applications.
  This includes ability to request new certificates, modify existing
  certificates, automatically renew certificates, and delete certificates. CPS
  also manages key Transport Layer Security (TLS) configurations, including
  cipher selection.
