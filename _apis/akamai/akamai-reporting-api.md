description: >-
  If youre using Akamai Intelligent Platform to deliver your content, you want
  to see how its performing. The Reporting API provides a wide range of reports,
  with new reports added periodically, and allows you to retrieve data in a
  range of intervals, from five minutes to monthly, depending on the time period
  and type of data you want to view. Some reports are available only to those
  who have purchased the related product. Support for specific intervals,
  filters, and metrics may vary by report type.
