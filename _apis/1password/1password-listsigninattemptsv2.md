- Sign-In Attempts

properties: []
description: >-
  Returns information about sign-in attempts to 1Password accounts. Each record includes the name and IP address of the user who attempted to sign in, when the attempt was made, the type of sign-in, the country, and for failed attempts the cause of failure. The request must include either a cursor from a previous response or a ResetCursor object with a start time.
