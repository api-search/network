description: >-
  This API endpoint retrieves analytics data about viewers for a specific piece of Confluence content identified by its contentId. When called with a GET request, it returns information about users who have viewed the specified content, which can be useful for understanding content engagement and reach within your Confluence space. The endpoint is part of the Confluence Analytics REST API and requires appropriate authentication and permissions to access viewer statistics for the requested content.
