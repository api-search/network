description: >-
  The Microsoft Azure Bare Metal Infrastructure Client is a service that
  provides customers with direct access to physical servers in Azure data
  centers, allowing for the deployment of virtual machines and applications
  without the need for a hypervisor. This service offers high performance,
  scalability, and security by providing dedicated hardware resources and a
  fully isolated environment for each customer. With Azure Bare Metal
  Infrastructure Client, customers can easily deploy and manage their workloads
  on bare metal servers, ensuring maximum control and flexibility while
  leveraging the benefits of the Azure cloud platform.
