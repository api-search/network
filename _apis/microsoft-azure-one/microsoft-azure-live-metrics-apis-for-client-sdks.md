  Microsoft Azure Live Metrics Rest APIs for Client SDKs allows developers to
  easily access and monitor real-time performance metrics of their applications
  running on Azure. By leveraging these APIs, developers can retrieve
  information such as CPU usage, memory utilization, and network latency, giving
  them valuable insights into the health and performance of their applications.
  This data can then be used to optimize and troubleshoot applications in
  real-time, ensuring a smooth and seamless user experience. With the power of
  Live Metrics Rest APIs, developers can proactively identify and address issues
  before they impact end-users, ultimately leading to better overall performance
  and reliability of their Azure applications.
