description: >-
  Updates storefront status-related settings.   - Channel ID can be used as a query parameter for updating channel-specific setting. If omitted, you will interact with the global setting only.    - `null` should be supplied to delete overrides per given channel and to inherit values from global level. Partial updates are not supported and all settings should be supplied with `null` value in order to delete overrides per channel.
