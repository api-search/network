description: >-
  Executes a specific API operation or workflow through the Jentic platform. Credentials for the upstream API are injected server-side by Jentic and are never sent to or stored by the agent. The operation is identified by its UUID, and the required inputs should match the schema returned by the load endpoint. Supports both single API operations and multi-step Arazzo workflows.
