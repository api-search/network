description: >-
  The Microsoft Azure Web Pub Sub Management Client is a tool that allows users
  to easily manage and monitor their Web Pub Sub service on the Azure cloud
  platform. With this client, users can create, update, and delete instances of
  the Web Pub Sub service, as well as configure settings, view metrics, and
  troubleshoot any issues that may arise. The Management Client provides a
  user-friendly interface for interacting with the Web Pub Sub service, making
  it simple to stay on top of the performance and availability of real-time
  messaging and data streaming applications.
