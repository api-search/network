description: >-
  Microsoft Azure VMware Cloud Simple is a service that allows customers to run
  VMware workloads natively on Azure, utilizing the familiar VMware stack
  without having to refactor applications. This service provides the flexibility
  and scalability of the Azure cloud infrastructure while maintaining the
  reliability and security of the VMware platform. With Azure VMware Cloud
  Simple, customers can easily migrate and extend their existing VMware
  environments to the cloud, enabling seamless integration with other Azure
  services and resources. This allows organizations to take advantage of the
  benefits of both VMware and Azure, creating a powerful and comprehensive
  hybrid cloud solution.
