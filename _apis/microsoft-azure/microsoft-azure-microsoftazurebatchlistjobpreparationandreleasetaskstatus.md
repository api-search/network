description: >-
  This API returns the Job Preparation and Job Release Task status on all Compute<br>Nodes that have run the Job Preparation or Job Release Task. This includes<br>Compute Nodes which have since been removed from the Pool. If this API is<br>invoked on a Job which has no Job Preparation or Job Release Task, the Batch<br>service returns HTTP status code 409 (Conflict) with an error code of<br>JobPreparationTaskNotSpecified.
