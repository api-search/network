description: >-
  Microsoft Graph Domains is the set of Microsoft Graph APIs and resources for
  discovering and managing your organizations domain names in Microsoft Entra ID
  (Azure AD). It lets you list all domains in a tenant and read key properties
  (for example, whether a domain is verified, default, initial, root, or which
  services it supports), add new custom domains, retrieve the DNS records needed
  for ownership verification, and complete the verification process. You can
  also manage authentication settings (managed or federated), work with
  federation configurations for SSO, and delete domains when theyre no longer
  referenced. These capabilities enable automation for tenant provisioning,
  domain onboarding and migration, and consistent domain governance across
  Microsoft 365 services.
