description: >-
  Microsoft Graph Agreement Acceptances provides a read-only way to retrieve
  records of users responses to your organizations Terms of Use configured in
  Microsoft Entra ID. Each acceptance entry is created automatically when a user
  (including guests) is prompted and captures who responded, which agreement and
  file version they saw, their response state (accepted or declined), the
  timestamp, andif perdevice consent is requiredthe device information. Through
  Microsoft Graph you can list and filter these records (for example by userId
  or agreementId) to build compliance and audit reports, verify who accepted
  which terms and when, and integrate this evidence into governance, automation,
  or SIEM workflows.
