description: >-
  The Fuel Prices API provides information about the current price of the fuel
  available at the selected station. The data is refreshed every 10 minutes, but
  the frequency of change can differ from a few times a day to a few times a
  month depending on the country or the brand of the gas station. Using this API
  makes it possible to make better price-conscious decisions while selecting a
  suitable gas station.
