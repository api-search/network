description: >-
  The Parking Availability API provides information about the current
  availability status of parking sites. The data is refreshed every 10 minutes
  and is therefore close to real time. Using this API makes it possible to make
  better off-street parking decisions based on actual parking availability
  (including the number of free spaces).
