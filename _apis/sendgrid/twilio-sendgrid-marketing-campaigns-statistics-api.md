  applications. You can also retrieve Marketing Campaigns stats in the
  [Marketing Campaigns application user interface](https://mc.sendgrid.com/).


  This API provides statistics for Marketing Campaigns only. For stats related
  to event tracking, please see the [Stats
  API](https://docs.sendgrid.com/api-reference/stats).
