  The GitHub User API endpoint `/enterprise/stats/users` using the GET method retrieves statistical information about users within a GitHub Enterprise instance. This endpoint provides aggregate data and metrics related to user accounts, which may include information such as total user counts, active users, user growth trends, and other relevant user-related statistics that help administrators monitor and analyze user engagement and activity across their enterprise organization.
