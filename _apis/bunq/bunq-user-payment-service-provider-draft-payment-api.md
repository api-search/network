  enables users to create drafts for payments within the Bunq banking platform.
  This allows users to easily initiate payments and send them for approval
  before they are finalized. With this API, users can draft payments for various
  purposes, such as paying bills, sending money to friends or family, or making
  purchases online. The API streamlines the payment process, making it more
  efficient and convenient for users to manage their finances through the Bunq
  platform.
