description: >-
  The Users API allows you to query and modify the state of your Connected Users
  and their integrations. The API includes REST endpoints (and matching SDK
  functions) for identifying what integrations your user has enabled,
  disconnecting integrations, and disabling workflows. The API also allows your
  application to associate metadata with a Connected User.
