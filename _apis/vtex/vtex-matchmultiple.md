description: >-
  Allows a marketplace to bulk approve, deny, or associate up to 25 received SKUs from sellers.  Through the `actionName` attribute, you can select the operation you want to apply to the received SKU. Actions include:   * `newproduct`: match the SKU as a new product.   * `skuassociation`: associate the received SKU to an existing SKU.   * `productassociation`: associate the received SKU to an existing product.   * `deny`: deny the received SKU.
