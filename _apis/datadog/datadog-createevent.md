description: >-
  Creates a new event in the Datadog Events Explorer. Events are useful for correlating infrastructure changes with metric anomalies. Common use cases include deployment markers, configuration changes, scaling events, and custom alerting from external systems. Events have a title, text body, priority, optional host and tag associations, and an alert type for visual categorization. The event timestamp defaults to the current time if not specified.
